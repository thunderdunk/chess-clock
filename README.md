# Chess Clock

A two-player countdown chess clock for mobile and tablet. Runs entirely in the browser — no install, no backend, no dependencies.

**Live:** https://chess-clock.dpsweet.workers.dev/

## Usage

Open the app in a browser or visit the link above. No install required.

- Place the device flat on the table between two players (portrait mode)
- Tap your side of the screen to stop your clock and start your opponent's
- Use the control strip in the center to start, pause, or reset the game

## Features

- Portrait/opposed layout — each player reads their clock right-side up
- Circular arc indicator showing remaining time at a glance
- Preset time controls: 1, 3, 5, and 10 minutes
- Low-time warnings with color shifts at 10 seconds
- Move counter
- Sound (Web Audio API, no external files) and haptic feedback
- Screen wake lock to prevent sleep during a game
